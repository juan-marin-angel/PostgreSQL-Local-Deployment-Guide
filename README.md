# Guía de Despliegue Local de PostgreSQL & pgAdmin con Docker

## Introducción

Este repositorio contiene una guía completa y práctica para desplegar un entorno local de PostgreSQL 16 y pgAdmin 4 utilizando Docker y Docker Compose.

El objetivo es facilitar la migración, restauración y prueba de bases de datos en un entorno aislado, reproducible y apto para desarrollo o validación de cambios antes de llegar a producción.

El contenido se basa en una necesidad real identificada durante mi trabajo como analista de automatización, en situaciones donde era necesario replicar estructuras de bases de datos remotas sin exponer datos sensibles.

## Contenido del Repositorio

El repositorio está dividido en dos enfoques principales: despliegue manual y despliegue automático, para que puedas elegir la metodología según tus necesidades.
